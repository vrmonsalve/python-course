# 📦 Module 06 — Libraries (numpy & pandas)

## What is a Library?

A library is a **collection of pre-built code** that you can import and use in your program. Instead of writing everything from scratch, you use what others have already built.

---

## numpy

- Stands for **Numerical Python**
- Used for **mathematical operations on arrays**
- Much faster than regular Python lists for numbers
- Foundation of most data science libraries

```bash
pip install numpy
```

---

## pandas

- Used for **data manipulation and analysis**
- Think of it as **Excel but in Python**
- Works with tables (called DataFrames)
- Can read CSV, Excel, JSON, and more

```bash
pip install pandas
```

---

## Real Life Uses

| Library | Use Case |
|---------|----------|
| numpy | Image processing, scientific computing, ML math |
| pandas | Data cleaning, analysis, reading spreadsheets |

---
