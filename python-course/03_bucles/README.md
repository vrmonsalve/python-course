# 🔁 Module 03 — Loops (for / while)

## What are Loops?

Loops let you **repeat a block of code** multiple times without writing it over and over.

```python
# Without a loop:
print("Hello")
print("Hello")
print("Hello")

# With a loop:
for i in range(3):
    print("Hello")
```

---

## Types of Loops

### for loop
Used when you know **how many times** to repeat.

```python
for item in collection:
    # do something with item
```

### while loop
Used when you repeat **until a condition is False**.

```python
while condition:
    # keep running
```

---

## Real Life Uses

- **for** — send an email to each user in a list
- **for** — process each row in a spreadsheet
- **while** — keep asking for input until it is valid
- **while** — run a game until the player quits
