# 🔀 Module 02 — Conditionals (if / elif / else)

## What are Conditionals?

Conditionals let your program **make decisions**. Based on a condition being True or False, the program takes different paths.

```python
if it_is_raining:
    take_umbrella()
else:
    wear_sunglasses()
```

---

## Syntax

```python
if condition:
    # runs if condition is True
elif other_condition:
    # runs if first was False and this is True
else:
    # runs if all conditions were False
```

---

## Comparison Operators

| Operator | Meaning |
|----------|---------|
| `==` | Equal to |
| `!=` | Not equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater or equal |
| `<=` | Less or equal |

---

## Real Life Uses

- **Login systems** — check if password is correct
- **E-commerce** — apply discount if user has coupon
- **Games** — check if player has won or lost
- **Validation** — check if a form field is empty
