## 🚀 How to Use This Repository

1. **Clone the repo**
```bash
git clone https://github.com/your-username/python-course.git
cd python-course
```

2. **Make sure Python is installed**
```bash
python --version
```

3. **Go through each module in order**
```bash
cd 01_variables
python variables.py
```

4. **Read the README.md in each folder** before running the code.

5. **Try the projects** once you finish the modules.
