# 🗃️ Module 05 — Data Structures

## What are Data Structures?

Data structures let you **store and organize multiple values** in a single variable.

---

## The Four Main Structures

| Structure | Syntax | Ordered | Mutable | Duplicates |
|-----------|--------|---------|---------|------------|
| List | `[1, 2, 3]` | ✅ | ✅ | ✅ |
| Tuple | `(1, 2, 3)` | ✅ | ❌ | ✅ |
| Dictionary | `{"key": value}` | ✅ | ✅ | Keys: ❌ |
| Set | `{1, 2, 3}` | ❌ | ✅ | ❌ |

---

## When to Use Each

- **List** — ordered collection you need to modify (shopping cart, student list)
- **Tuple** — fixed data that should not change (coordinates, RGB colors)
- **Dictionary** — key-value pairs (user profile, product catalog)
- **Set** — unique values, fast membership testing (tags, categories)
