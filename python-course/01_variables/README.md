# 📦 Module 01 — Variables & Data Types

## What is a Variable?

A variable is a **container that stores data**. Think of it like a labeled box where you put information so you can use it later.

```python
name = "Maria"   # box labeled "name" containing "Maria"
age = 25         # box labeled "age" containing 25
```

---

## Data Types

| Type | Example | Used for |
|------|---------|----------|
| `str` | `"Hello"` | Text |
| `int` | `42` | Whole numbers |
| `float` | `3.14` | Decimal numbers |
| `bool` | `True / False` | Yes/No values |

---

## Why Does This Matter in Real Life?

- A shopping app stores your **name** (str), **age** (int), and **balance** (float)
- A login system checks if a user **is authenticated** (bool)
- Every program you've ever used relies on variables

## How to Run

python variables.py

## Key Takeaways

- Variables store data using `=`
- Python automatically detects the type
- Use `type()` to check the type of any variable
- Variable names should be clear and descriptive
